Quantum error correction & fault tolerance

Quantum Computing · June 2026 · after D. Gottesman, arXiv:0904.2557

Download PDF ↓ LaTeX source ↓

Quantum states are delicate, so building a useful quantum computer means protecting fragile qubits with error correction and then computing on the encoded data without that protection unravelling. This synopsis follows Gottesman's review: why quantum error correction looks impossible at first, the nine-qubit code that makes it possible, the conditions a code must satisfy, the stabilizer formalism and its links to classical coding theory, and then the harder problem of fault tolerance — computing reliably with components that are themselves noisy — culminating in the threshold theorem.

1. Why it looks impossible

Three obstacles set the quantum problem apart from the classical one.

No cloning. The classical repetition code copies a bit, \(0\to000\), and corrects by majority vote. The quantum analogue \(\ket{\psi}\to\ket{\psi}\ket{\psi}\ket{\psi}\) is forbidden:

No-cloning theorem. No quantum operation sends \(\ket{\psi}\to\ket{\psi}\ket{\psi}\) for all \(\ket{\psi}\). If it did, linearity would force \(\ket{\psi}+\ket{\phi}\mapsto\ket{\psi}\ket{\psi}+\ket{\phi}\ket{\phi}\), which disagrees with the definition's \((\ket{\psi}+\ket{\phi})(\ket{\psi}+\ket{\phi})\) by the cross terms \(\ket{\psi}\ket{\phi}+\ket{\phi}\ket{\psi}\).

Continuous errors. Qubit errors form a continuum: an over-rotation sends \(\alpha\ket0+\beta e^{i\phi}\ket1\) to \(\alpha\ket0+\beta e^{i(\phi+\delta)}\ket1\), wrong by an arbitrarily small \(\delta\) that accumulates. We will need to digitize these errors.

Decoherence. Measuring \(\alpha\ket0+\beta\ket1\) collapses it to \(\ket0\) or \(\ket1\); the environment is constantly trying to "look." Error correction must extract information about the error without learning anything about the encoded data.

2. The nine-qubit code and error digitization

Shor's code nests two repetition codes. Encode

\[ \ket{\bar0} = \tfrac{1}{2\sqrt2}\big(\ket{000}+\ket{111}\big)^{\otimes 3},\qquad \ket{\bar1} = \tfrac{1}{2\sqrt2}\big(\ket{000}-\ket{111}\big)^{\otimes 3}. \]

An arbitrary codeword \(\alpha\ket{\bar0}+\beta\ket{\bar1}\) is a superposition, so no-cloning is not violated; the codewords span a linear coding subspace. The inner triplets correct bit flips by majority, \(\ket{010}\pm\ket{101}\to\ket{000}\pm\ket{111}\); the outer layer corrects phase flips by majority on the three signs. Crucially, parities are measured so as to reveal the error and nothing else.

The four single-qubit operators \(I,X,Z,Y=iXZ\) (bit flip, phase flip, both) generate the Pauli group \(\mathcal P_n\): tensor products of \(I,X,Y,Z\) with a phase \(\pm1,\pm i\), so \(4^{n+1}\) elements. Any two elements commute or anticommute, and \(XZ=-ZX\). The weight \(\mathrm{wt}(Q)\) is the number of non-identity tensor factors; the \(n\)-qubit Paulis span all \(2^n\times 2^n\) matrices.

That spanning property is what digitizes errors. A continuous phase error \(R_{\theta/2}=\cos\tfrac\theta2\,I - i\sin\tfrac\theta2\,Z\) is corrected with no extra effort: recording the error type in an ancilla and measuring it projects the continuous error onto either "no error" (apply \(I\)) or "\(Z\) error" (apply \(Z\)), each of which the code already handles. This generalizes:

Linearity of correction. If a code corrects errors \(A\) and \(B\), it corrects any linear combination \(\alpha A+\beta B\). In particular, a code that corrects all weight-\(t\) Pauli errors corrects all errors on \(t\) qubits, including general decoherence \(\rho\to\sum_i E_i\rho E_i^\dagger\): each \(E_i\) acting on \(t\) qubits lies in the span of \(t\)-qubit Paulis.

3. When does a subspace correct errors?

We never want to confuse \(\ket{\bar0}\) with \(\ket{\bar1}\), even after errors: \(E\ket{\bar0}\) must stay orthogonal to \(F\ket{\bar1}\). Distinguishing the error is sufficient but not necessary — the nine-qubit code cannot tell \(Z_1\) from \(Z_2\), yet corrects either. Massaging these requirements gives the Knill–Laflamme conditions.

QECC condition. A subspace \(C\) corrects a linear space of errors \(\mathcal E\) iff for all \(E\in\mathcal E\) and all codewords,

\[ \bra{\psi}E^\dagger E\ket{\psi} = C(E), \quad\text{independent of }\ket\psi; \qquad\text{equivalently}\quad \bra{\psi_i}E_a^\dagger E_b\ket{\psi_j} = C_{ab}\,\delta_{ij}. \]

If the Hermitian matrix \(C_{ab}\) has maximal rank the code is nondegenerate; if not (as for Shor's code, where different errors act identically on the code) it is degenerate. Reading the condition with \(E\) on \(2t\) qubits says no measurement on \(2t\) qubits can learn anything about the codeword — equivalently, one can detect up to \(2t\) errors.

A code correcting \(t\) errors has distance \(d=2t+1\) (the minimum weight to map one codeword to another); using \(n\) physical qubits to encode \(k\) logical qubits at distance \(d\) gives an \([[n,k,d]]\) code. Counting states bounds the parameters:

\[ \underbrace{\Big(\textstyle\sum_{j=0}^{t} 3^j\binom{n}{j}\Big)2^k \le 2^n}_{\text{quantum Hamming bound (nondegenerate)}},\qquad \underbrace{n-k \ge 2d-2}_{\text{quantum Singleton / Knill--Laflamme bound}}. \]

A matching quantum Gilbert–Varshamov bound guarantees codes exist when an analogous sum up to \(d-1\) holds. The five-qubit \([[5,1,3]]\) code saturates the Singleton bound, so it is the smallest code correcting one arbitrary error; the nine-qubit code is \([[9,1,3]]\).

4. The stabilizer formalism

Return to the nine-qubit code: the corrective parity checks are exactly measurements of commuting Pauli operators like \(Z_1Z_2\) and \(X_1\cdots X_6\). The group they generate is the stabilizer \(S\) — an Abelian subgroup of \(\mathcal P_n\) with \(-1\notin S\), defined so that

\[ M\ket{\psi}=\ket{\psi}\quad\text{for all } M\in S,\ \ket{\psi}\in C. \]

The magic is in the syndrome: if an error \(E\) anticommutes with a generator \(M\), then

\[ M\,(E\ket{\psi}) = -E M\ket{\psi} = -E\ket{\psi}, \]

so the eigenvalue flips from \(+1\) to \(-1\). Measuring the generators yields the error syndrome — the list of which generators anticommute with the error — which pins down \(E\) (up to the degeneracy \(E^\dagger F\in S\)). Conversely, any Abelian \(S\) with \(-1\notin S\) defines a code:

Stabilizer code theorem. If \(S\) is an Abelian subgroup of order \(2^a\) of \(\mathcal P_n\) with \(-1\notin S\), and \(d\) is the minimum weight of an operator in \(N(S)\setminus S\), then the states fixed by \(S\) form an \([[n,n-a,d]]\) code. Here \(N(S)\) is the normalizer (the Paulis commuting with \(S\)).

Stabilizers also have a compact binary symplectic representation: write each Pauli as a pair \((a\mid b)\) of binary vectors (a \(1\) in \(a\) for \(X\) or \(Y\), a \(1\) in \(b\) for \(Z\) or \(Y\)). Multiplication is addition, and two operators commute iff \(a_1\!\cdot\!b_2 + b_1\!\cdot\!a_2 = 0\) — a symplectic inner product. The stabilizer condition becomes "self-orthogonal under the symplectic form."

5. A zoo of codes

CSS codes. Take two classical linear codes with \(C_2^\perp\subseteq C_1\) so that the \(X\)- and \(Z\)-type stabilizer generators commute. The result is an \([[n,\,k_1+k_2-n,\,\min(d_1,d_2)]]\) code (Calderbank–Shor–Steane). Its codewords have the clean form

\[ \ket{\overline{u}} = \sum_{w\in C_2^\perp}\ket{u+w},\qquad u\in C_1, \]

so \(Z\)-parities are checked in the standard basis and \(X\)-parities in the Hadamard basis. The seven-qubit Steane code \([[7,1,3]]\) uses the classical Hamming code for both, which makes its logical Clifford gates especially clean.

Codes over GF(4). Identifying \(I,Z,X,Y\) with \(0,1,\omega,\omega^2\) in the four-element field \(\mathrm{GF}(4)\) turns the commutativity constraint into a trace-Hermitian self-orthogonality of a classical \(\mathrm{GF}(4)\) code. The five-qubit code is just a Hamming code over \(\mathrm{GF}(4)\); the entire toolbox of classical coding theory becomes available.

Beyond stabilizers. Qudit codes generalize to higher-dimensional registers; subsystem (operator) codes ignore some degrees of freedom and help with fault tolerance; decoherence-free subspaces are fully degenerate codes needing no active correction but only for special noise; and dynamical decoupling cancels slowly varying noise with fast pulses and no extra qubits. In practice one layers these: decoupling first, then a tailored code, then a general QECC — each layer paid for in extra qubits and extra errors to clean up.

6. Logical operators and the Clifford group

The normalizer modulo the stabilizer is the logical Pauli group: \(N(S)/S \cong \mathcal P_k\). An operator \(E\in N(S)\) commutes with all of \(S\), so it preserves the code space; if \(E\notin S\) it acts as a nontrivial logical operation \(\bar X,\bar Z\). Two Paulis with the same syndrome differ by an element of \(S\), and \(\lvert N(S)/S\rvert = 4^{k+1}\).

The Clifford group \(\mathcal C_n\) is the normalizer of \(\mathcal P_n\) in the unitary group: \(U P U^\dagger\in\mathcal P_n\) for all \(P\). It is generated by \(H\), the \(\pi/4\) phase gate \(P=\mathrm{diag}(1,i)\), and CNOT, and a Clifford is fixed (up to phase) by how it permutes \(X\) and \(Z\), e.g. \(H:X\mapsto Z,\ Z\mapsto X\). Since Cliffords correspond to symplectic matrices, \(\mathcal C_n/\mathcal P_n' \cong \mathrm{Sp}(2n,\mathbb Z_2)\). One consequence is the Gottesman–Knill theorem: a circuit of Clifford gates on a stabilizer state with Pauli measurements is efficiently classically simulable — so Cliffords alone are not universal.

7. Fault tolerance: the idea

Encoding is not enough: we must compute on encoded data, and correct errors, using gates that are themselves noisy. A protocol is fault tolerant if a small number of component failures produces only a correctable number of errors. The basic locations are preparation, gates from a universal set \(\{H,\text{CNOT},R_{\pi/8}\}\) with \(R_{\pi/8}=\mathrm{diag}(1,e^{i\pi/4})\), measurement, and wait steps; a faulty location can corrupt all qubits it touches.

The central danger is error propagation. Even a perfect gate spreads existing errors:

\[ U E\ket{\psi} = (UEU^\dagger)\,U\ket{\psi}, \qquad \text{CNOT}: X\otimes I\mapsto X\otimes X,\ \ I\otimes Z\mapsto Z\otimes Z. \]

A two-qubit gate can turn one error into two, overwhelming a distance-3 code. Fault-tolerant gadgets are designed so this never happens. Rigorously (Aliferis–Gottesman–Preskill), correctness is phrased with an \(r\)-filter (projector onto states \(Q\ket{\psi}\) with \(\mathrm{wt}(Q)\le r\)) and an ideal decoder. A gadget is fault tolerant if, whenever input errors plus gadget faults total at most \(t\): (A) errors do not propagate too far within a block, and (B) it still performs the correct logical operation. These yield precise definitions of fault-tolerant preparation, measurement, gates, and error correction.

8. Transversal gates and their limit

The cleanest fault-tolerant gates are transversal: a tensor product \(U=\bigotimes_i U_i\) where the \(i\)-th qubit of one block interacts only with the \(i\)-th qubit of another. Such a gate propagates an error to at most one qubit per block, so it is automatically fault tolerant. Logical Paulis are transversal for any stabilizer code; for the seven-qubit code, \(H^{\otimes 7}=\bar H\), \(P^{\dagger\,\otimes 7}=\bar P\), and transversal CNOT \(=\overline{\text{CNOT}}\) (the latter for any CSS code), giving the whole logical Clifford group transversally.

But Cliffords are not universal, and:

Eastin–Knill. No quantum code admits a universal set of transversal logical gates. Some non-transversal construction is unavoidable to finish a universal set.

9. Fault-tolerant error correction

Measuring a stabilizer generator naively (one ancilla, controlled-Pauli to every data qubit) lets a single ancilla error propagate to many data qubits. Three schemes fix this.

Shor EC. Spread the ancilla into a verified \(n\)-qubit cat state \(\ket{0\cdots0}+\ket{1\cdots1}\), so the \(i\)-th data qubit interacts with only the \(i\)-th ancilla qubit. Measuring the ancilla in the Hadamard basis and taking the parity gives one syndrome bit. Because a single fault can corrupt a syndrome bit, the whole syndrome is measured repeatedly and a consensus taken. Robust but laborious.

Steane EC. For CSS codes, prepare ancilla blocks in encoded \(\ket{\bar0}+\ket{\bar1}\) and \(\ket{\bar0}\), apply transversal CNOT between data and ancilla, and measure the ancilla. Bit-flip errors copy onto one ancilla (measured to give \(C_1\) syndrome \(e+f\)), phase-flip errors onto the other (giving \(C_2\) syndrome) — transversality keeps a single measurement fault to a single bit.

Knill EC. Teleport the encoded state into a fresh block; the encoded Bell measurement reveals the combined error syndrome as a byproduct. Works for any stabilizer code. Steane and Knill EC both push effort into preparing known ancilla states (which can be verified offline) rather than touching the unknown data — the source of their high thresholds.

10. Completing universality: magic states

Since \(R_{\pi/8}\) is not transversal, it is supplied by gate teleportation through a special ancilla. Teleporting a state and then applying \(\bar U\) equals teleporting through the ancilla \((I\otimes\bar U)(\ket{\overline{00}}+\ket{\overline{11}})\) with a modified correction \(\overline{UQU^\dagger}\). The construction works whenever \(UQU^\dagger\) is easier than \(U\) — which happens for \(R_{\pi/8}\), since

\[ R_{\pi/8}\,X\,R_{\pi/8}^\dagger = e^{i\pi/4} X P^\dagger \in \mathcal P_1\cdot\mathcal C_1, \qquad R_{\pi/8}\,Z\,R_{\pi/8}^\dagger = Z. \]

This defines the Clifford hierarchy \(\mathcal C_k = \{U : U\mathcal P U^\dagger \subseteq \mathcal C_{k-1}\}\): \(\mathcal C_1\) is the Paulis, \(\mathcal C_2\) the Cliffords, and \(R_{\pi/8}\in\mathcal C_3\). Gates in \(\mathcal C_k\) are teleported using ancillas prepared with \(\mathcal C_{k-1}\) gates. So a transversal Clifford group plus a single verified magic state for \(R_{\pi/8}\) gives a universal fault-tolerant set on the seven-qubit code. The finite set \(\{H,\text{CNOT},R_{\pi/8}\}\) is dense in \(U(2^n)\), and by Solovay–Kitaev any gate is approximated to accuracy \(\epsilon\) with \(\mathrm{polylog}(1/\epsilon)\) gates.

11. The threshold theorem

With fault-tolerant gadgets for every location, one studies extended rectangles (ExRecs): a gate gadget together with the leading and trailing error-correction steps. An ExRec is good if it contains at most \(t\) faults (\(t\) = the code's correction capacity), and:

Good implies correct. A good ExRec performs the right logical operation; a circuit of only good ExRecs reproduces the ideal circuit (Theorems 6–8, with truncated ExRecs and \(*\)-decoders to handle overlaps).

The payoff comes from concatenation: encode the code inside itself \(L\) times. Under a local stochastic error model (probability \(\le p^r\) of faults on any \(r\) fixed locations) the logical error rate per level obeys

\[ p' \le A\,p^{\,t+1}, \qquad\text{define}\quad p_T = 1/A^{1/t}, \qquad\text{then}\quad \frac{p^{(j)}}{p_T} \le \Big(\frac{p}{p_T}\Big)^{(t+1)^j}. \]

Concrete numbers depend on the code and analysis: a careful seven-qubit-code accounting gives \(p_T\gtrsim 2.73\times10^{-5}\); the best rigorous lower bound is around \(10^{-3}\); and Knill's teleportation-based schemes simulate thresholds as high as a few percent. The theorem needs parallel operations (you cannot let qubits wait uncorrected), the ability to prepare fresh ancillas mid-computation, non-destructive errors, and bounded error correlations (the local stochastic, or more generally non-Markovian, assumption).

Takeaways

  • No-cloning and continuous errors are evaded by encoding in a linear subspace and digitizing: correcting all weight-\(t\) Paulis corrects all \(t\)-qubit errors.
  • The Knill–Laflamme conditions say exactly when a subspace is a code; distance \(d=2t+1\) and the Hamming / Singleton bounds constrain \([[n,k,d]]\).
  • The stabilizer formalism turns codes into Abelian Pauli groups, syndromes into eigenvalue flips, and connects quantum codes to classical and \(\mathrm{GF}(4)\) coding theory.
  • Fault tolerance is about taming error propagation; transversal gates are ideal but never universal (Eastin–Knill).
  • Universality is completed by teleporting magic states up the Clifford hierarchy.
  • Below a constant threshold, concatenation suppresses logical error doubly exponentially with polylogarithmic overhead — arbitrarily reliable quantum computation is possible.
